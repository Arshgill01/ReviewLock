# ReviewLock

ReviewLock is a Reddit Devvit moderation app that locks reviewed content until it changes.

This repository is built from the wave plan in `plan.md`. Coding agents should read `AGENTS.md`, `plan.md`, `docs/OWNERSHIP.md`, and the active `waves/wave-XX/spec.md` before editing.

## Development

```bash
npm install
npm run type-check
npm run lint
npm run test
npm run build
```

Runtime behavior is not claimed as verified until documented in `docs/RUNTIME_PROOF.md`.
